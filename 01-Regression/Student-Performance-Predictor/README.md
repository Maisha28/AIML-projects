# Student Performance Predictor

A simple machine learning project that predicts a student's final grade using academic and personal factors.
This project was built to learn the complete machine learning workflow, including data preprocessing, feature selection, model training, evaluation, model saving with Pickle, and deployment using Streamlit.

### Tech Stack

* Python
* Pandas
* NumPy
* Scikit-Learn
* Streamlit

### Model

* Algorithm: Linear Regression
* R² Score: 0.77

### Features Used

* Parents' education level
* Study time
* Travel time
* Previous failures
* Internet access
* Absences
* G1 and G2 grades
* Other student-related factors

### Run Locally
bash
pip install -r requirements.txt
python train.py
streamlit run app.py
